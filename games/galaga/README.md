# 🚀 갤러그 (Galaga) - 클래식 아케이드 게임

1981년 출시된 전설적인 아케이드 슈팅 게임 갤러그를 HTML5 Canvas와 JavaScript로 재현한 프로젝트입니다.

## 게임 특징

### 🎮 게임플레이
- **클래식 슈팅 게임**: 원작 갤러그의 게임플레이를 충실히 재현
- **다양한 적 타입**: 보스(빨강), 나비(파랑), 벌(노랑) 3가지 타입
- **급강하 공격**: 적들이 플레이어를 향해 급강하 공격
- **레벨 시스템**: 레벨이 올라갈수록 더 많은 적 출현
- **생명 시스템**: 3개의 생명으로 시작
- **점수 시스템**: 적 타입에 따라 다른 점수 획득
  - 보스: 150점
  - 나비: 80점
  - 벌: 50점

### 🎯 게임 기능
- 최고 점수 저장 (로컬 스토리지 사용)
- 일시정지 기능
- 별이 흐르는 우주 배경
- 폭발 효과
- 반응형 UI

## 조작법

| 키 | 기능 |
|---|---|
| ← (왼쪽 화살표) | 좌측 이동 |
| → (오른쪽 화살표) | 우측 이동 |
| 스페이스바 | 발사 |
| P | 일시정지 |

## 실행 방법

### 웹 브라우저에서 직접 실행

1. `index.html` 파일을 웹 브라우저로 열기
   ```bash
   # Chrome에서 열기 (Linux)
   google-chrome index.html

   # Firefox에서 열기 (Linux)
   firefox index.html
   ```

2. 또는 파일 탐색기에서 `index.html` 더블클릭

### 로컬 서버로 실행 (권장)

Python 3가 설치되어 있다면:

```bash
cd games/galaga
python -m http.server 8000
```

그런 다음 브라우저에서 `http://localhost:8000` 접속

Node.js가 설치되어 있다면:

```bash
cd games/galaga
npx serve .
```

## 게임 구조

### 주요 클래스

1. **Player**: 플레이어 우주선
   - 좌우 이동
   - 발사 기능
   - 총알 관리

2. **Enemy**: 적 우주선
   - 3가지 타입 (보스, 나비, 벌)
   - 좌우 이동 및 하강
   - 급강하 공격
   - 발사 기능

3. **Bullet**: 발사체
   - 플레이어와 적의 총알 구현
   - 각각 다른 색상과 방향

4. **Explosion**: 폭발 효과
   - 충돌 시 폭발 애니메이션

### 게임 상태

- `menu`: 시작 화면
- `playing`: 게임 진행 중
- `paused`: 일시정지
- `gameOver`: 게임 오버

## 게임 팁

1. **적 우선순위**: 보스를 먼저 처리하면 높은 점수 획득
2. **급강하 회피**: 적의 급강하 패턴을 익혀 회피
3. **발사 타이밍**: 쿨다운을 고려한 효율적인 발사
4. **위치 선정**: 화면 중앙보다는 측면에서 회피가 용이
